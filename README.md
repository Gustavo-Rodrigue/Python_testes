Este repositório tem como intuito abranger diversas situações em python que possam ajudar a pessoas em iniciação no python.

## 📕 Conteúdo

Neste repositório será possivel ver diversas atividades em python demonstrando diversas situações com possibilidades de resolução utilizando python.

### 📁 Arquivos do conteúdo

* **Decisao** - Esta pasta contém diversos arquivos com sistemas de decisão (if, elif e else).

* **Funções** - Esta pasta contém diversos arquivos com sistemas de funções pre-existentes e funções criadas (def e max()).

* **Operadores** - Esta pasta contém diversos arquivos coom testes de operadores (+, -, <, > e etc.).

* **Repetição** - Esta pasta contém diversos arquivos com estruturas de repetição (while e for).

* **Listas/Desafios** - Esta pasta contém listas de atividades e desafios realizados para de monstrar meus conhecimentos.

## 🛠️ Tecnologias Utilizadas

* [Python](https://www.python.org/): Linguagem principal.
* [Visual Studio Code](https://code.visualstudio.com/): Editor de códigos.
